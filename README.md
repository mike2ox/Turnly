# Turnly: 대화 발화 타이머

> 대화에서 내가 정해진 시간 안에 발화를 마치는지,  
> 그리고 제한 시간이 지난 후 얼마나 더 말하다가 넘기는지를 측정하는 도구입니다.

---

## 왜 만들었나

회의나 대화에서 발화 시간을 의식하기 어렵습니다. 내가 정해진 시간 안에 할 말을 다 했는지, 제한이 끝난 뒤에도 계속 말하고 있는 건 아닌지 — 그 감각을 숫자로 확인하고 싶어서 만든 도구입니다.

---

## 기능

### 세 개의 타이머

| 타이머         | 방식                            | 역할                            |
| -------------- | ------------------------------- | ------------------------------- |
| 전체 대화 시간 | 카운트다운                      | 세션 전체 남은 시간             |
| 발화 제한시간  | 아날로그 바늘 시계 + 카운트다운 | 내 발화에 주어진 시간           |
| 초과 타이머    | 카운트업                        | 제한 종료 후 얼마나 더 말했는지 |

발화 제한시간이 끝나는 순간 초과 타이머가 자동으로 시작됩니다. "대화 넘기기" 버튼을 누르는 순간까지 카운트업되며, 해당 값이 기록에 남습니다.

### 타임박싱 (아젠다 설정)

세션 시작 전에 대화 아이템을 미리 구성할 수 있습니다.

- 아이템명과 시간(분)을 입력해 목록 구성
- 세션 시작 시 아이템 순서대로 자동 진행
- 각 아이템의 시간이 발화 제한시간으로 적용
- "대화 넘기기" 버튼을 누르면 다음 아이템으로 자동 이동
- 모든 아이템 완료 시 세션 자동 종료
- 아이템이 없으면 수동 모드로 동작

### 기록

대화를 넘길 때마다 아이템명과 초과/여유 여부가 기록 탭에 쌓입니다.

---

## 사용 방법

**수동 모드 (아이템 없이)**

1. 설정 탭에서 전체 대화 시간과 발화 제한시간을 입력
2. 시작
3. 발화가 끝나면 "대화 넘기기" 클릭

**타임박싱 모드**

1. 타임박싱 탭에서 아이템 추가 (예: `여행지 브레인스토밍 / 20분`)
2. 설정 탭에서 시작
3. 각 아이템이 끝나면 "대화 넘기기" 클릭 → 자동으로 다음 아이템 시작

## 배포 (GitHub Pages)

```bash
# 1. 레포지토리 생성 (Public)
#    이름: Turnly

# 2. index.html 업로드

# 3. Settings → Pages → Branch: main / (root) → Save

# 4. https://{username}.github.io/Turnly/ 접속
```

별도 서버, 빌드 도구, 패키지 설치 없이 단일 HTML 파일로 동작합니다.

## 기술 스택

- Vanilla HTML / CSS / JavaScript (의존성 없음)
- [Tabler Icons](https://tabler.io/icons) — 아이콘
- [Google Fonts](https://fonts.google.com) — DM Mono, Noto Sans KR
- SVG 아날로그 시계 (Canvas 없이 순수 SVG + JS)

## 로컬 실행

```bash
git clone https://github.com/{username}/Turnly.git
cd Turnly
open index.html   # macOS
# 또는 index.html 파일을 브라우저로 드래그
```

인터넷 연결 없이도 완전히 동작합니다 (폰트만 외부 로드).
